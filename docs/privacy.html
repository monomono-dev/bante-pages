プライバシーポリシー

BAN.TE — ゴルフ練習サポートアプリ

1. データの取り扱い

BAN.TE は、練習記録・スイングメモ・持ち物チェックリストなどのデータを端末のローカルデータベースに保存します。基本的な利用にアカウント登録やログインは不要です。

2. Googleアカウントによるクラウドバックアップ

本アプリには、Googleアカウントでログインしてデータをクラウドにバックアップする機能があります（任意）。この機能を利用する場合、以下の情報が収集・保存されます。

  • Googleアカウントのメールアドレス（認証のために Firebase Authentication で使用）
  • 練習記録・ショットメモ・スイングメモ・持ち物リスト・設定情報（Google Cloud Firestore に暗号化通信で保存）

これらのデータはユーザー本人のみがアクセスでき、第三者への提供・販売は行いません。バックアップ機能を使用しない場合、データは端末内にのみ保存され、外部への送信は一切行われません。

3. 分析・トラッキング

本アプリにはアクセス解析ツールやトラッキングSDKを組み込んでいません。利用状況の追跡は一切行いません。

4. 外部リンク

本アプリには、持ち物リストの便利機能としてAmazon.co.jpへの検索リンクが含まれています。リンクをタップすると端末の標準ブラウザが起動します。

リンク先のサイトにおけるプライバシーについては、各サイトのポリシーをご確認ください。

5. データの削除

アプリを端末からアンインストールすると、端末内のすべてのデータが削除されます。

クラウドバックアップを利用した場合は、アプリ内の「マイページ」から「クラウドデータを削除」を選択することで、サーバー上のバックアップデータを完全に削除できます。
